## Overview

Meralco.py is a comprehensive billing and sub-metering application designed to streamline the management of electricity billing. The application is divided into two integrated modules:
- **Bill Detail Module:** Manages customer information, kWh readings, consumption calculations, and bill generation.
- **Sub-Metering Module:** Handles sub-meter readings, data filtering, and record management using CSV/JSON formats.

## Features

- **User-Friendly Interface:** Built with Pythonâ€™s Tkinter for an intuitive GUI experience.
- **Real-Time Consumption Calculation:** Automatically computes consumption differences based on previous and current kWh readings.
- **Data Import/Export:** Easily load and save billing data in CSV and JSON formats.
- **Record Filtering:** Utilize customizable filters for quick record searches.
- **Copy Options:** Copy billing data as text or as a generated image for reporting purposes.

## Installation

1. **Download the Executable:** Obtain the `Meralco.py` EXE file.
2. **Run the Application:** Double-click the executable to launch the application.
3. **Bundled Dependencies:** All necessary libraries are included with the EXE.

## Usage

- **Bill Detail Module:** Input customer details along with previous and current readings. The app calculates the total consumption and the corresponding bill based on the current rate.
- **Sub-Metering Module:** Manage additional meter readings. Use the filtering tools to view or load records, and save updated records to JSON.
- **Data Copying:** Use the provided buttons to copy billing data either as text or as an image for documentation or reporting purposes.
